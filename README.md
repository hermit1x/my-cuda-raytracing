# RayTracing in CUDA

跟着RayTracing in One Weekend学完之后感觉运算速度不够快，想着或许可以用CUDA进行加速，于是有了这么一个repo。

主要参考另一个一样使用cuda的项目，实现the first week，接着手动增加了next week的内容，第三部分的蒙特卡洛等特性还在施工中。

然后目前就算CUDA加速有很明显的提升，但对速度还是不是很满意，打算施工完成后参考参考中的3进行进一步加速调整。
